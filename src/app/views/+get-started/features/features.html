Open source container-native workflow engine built for Kubernetes

Develop and deploy applications on Kubernetes as simple and as reliable as possible. Run thousands of concurrent containerized workflows ultra-fast.

  • Each step in the workflow is a container
  • Arbitrarily compose sub-workflows to create larger workflows
  • Configuration as code (YAML for everything)
  • No need to learn tools such as Jenkins, Chef, Cloud Formation

Docker-in-Docker

  • Run DinD (Docker-in-Docker) out of the box
  • Run Docker builds and other containers from within containerized workflows

Built-in artifact management

  • Integrated artifact management for storing and passing artifacts
  • Easily "pass" and reference artifacts between steps in workflows

Built-in secret management

  • Define, store, and access secrets from workflows
