Argo is super simple to install and set up! Check these prerequisites
  • Make sure you have an AWS account with Minimum resource quota requirements for Argo running in AWS.
  • Docker client is installed on your client system.
  • AWS Command-Line Interface (CLI) is installed on your client system.
 before you start the installation.

For Mac
curl -sSL -O https://s3-us-west-1.amazonaws.com/ax-public/argocli/latest/darwin_amd64/argo

For Linux
curl -sSL -O https://s3-us-west-1.amazonaws.com/ax-public/argocli/latest/linux_amd64/argo

chmod +x argo

Then 
./argo cluster
to open the Argo Command Shell

Once you get the prompt, 
argo cluster ops> argocluster install
to start the installation

Then enter a name for your cluster and your AWS profile details. Your cluster would be ready in about 20 minutes.
